=== HueCentral Webapp Notes ===

-- HTML5 Features --
* LocalStorage
  - Hue bridge address
  - Hue bridge username
* Application Cache
* Geolocation (for daylight sensor)

-- App Features --
* Use Hue daylight sensor
  - Get geolocation from browser
  - Insert geolocation into Hue bridge
  - Setup rules based on sensor
* Create/edit rules for sensors
* Display "poweruser" information
* Press "link" button on bridge
